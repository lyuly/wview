# 介绍

## 我们是谁？

来自天南地北的一群前端爱好❤️者，因为[字节跳动青训营](https://youthcamp.bytedance.com/)而相识😆

## 为什么做这个？

提升自己，弥补不足，但更多的是因为对前端的热爱🥰

## 技术选型
* Monorepo架构
* Vite、Vue3、TypeScript、Sass、Vitepress
* Husky、lint-staged、eslint、prettier
* vercel
